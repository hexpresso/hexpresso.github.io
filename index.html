[ NEWS ][15/05/2015]

Hexpresso is now on Github !

We have decided to migrate to Github for many reasons:

  • Wordpress is not really pleasant to use,
  • Github is simpler to sort our write-ups by CTF,
  • Github allows us to create and modify very easily our WU, by merging using git,
  • People reading our WU will be able to clone our repo which might contains binaries, pictures, etc.
So now, we will slowly migrate all our WU from Wordpress to our Github.

[ Hexpresso story and composition ]

Quite often, we participate in both on-line and off-line security Capture The Flag competitions. After some of them, we publish write-ups on tasks that you can find on our Github.
